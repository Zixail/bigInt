# bigInt — Простая библиотека для работы с большими целыми

Репозиторий содержит реализацию работы с большими целыми числами на C. Основной файл с реализацией: [bigInt.c](bigInt.c). В проекте также есть файл игнорирования: [.gitignore](.gitignore).

## Описание
Программа реализует чтение больших целых в десятичной форме, внутреннее представление в виде массива 16‑битных слов и операции:
- Сложение — `sumBigInt`
- Вычитание — `subtractBigInt`
- Умножение — `multiBigInt`
- Возведение в степень — `powBigInt`
- Факториал — `factBigInt`
- Сумма прогрессии — `progrBigInt`

Вспомогательные функции: преобразование из/в десятичное представление (`toBinary`, `toDecimal`), сканирование/печать чисел (`scanBigInt`, `printBigInt`) и утилиты для работы с двоичным представлением (`getLength`, `toNegative`, `toPositive`, `shiftBinary`).

## Структура
- [bigInt.c](bigInt.c) — весь исходный код реализации (функции выше).
- [.gitignore](.gitignore) — правила игнорирования.

## Сборка
Компиляция GCC:
```sh
gcc bigInt.c -o bigInt.exe
```

## Запуск и использование
После сборки запустите:
```sh
./bigInt.exe
```
Программа ожидает сначала операцию (например: +, -, *, ^, !, S), затем одно или два числа в десятичной форме в зависимости от операции (`!` — только одно число). Выводится строка `Result=` и результат.

Примеры операций:
- Сложение: ввести `+` затем два числа.
- Факториал: ввести `!` затем одно число.

## Примечания и ограничения
- Реализация использует фиксированные размеры массивов: `SIZE_B`, `SIZE_HU` и `SIZE_S` в [bigInt.c](bigInt.c).
- Представление знака и операции реализованы вручную (дополнительный код и т.д.).
- Код предназначен для образовательных/учебных целей и не оптимизирован для производительности или безопасности ввода.
